# 河南牛价行情 PWA

河南省牛业数据实时查看应用

## 功能

- 活牛、牛肉、羊肉、猪肉等畜产品价格
- 价格走势图表
- 河南省各地存栏量排名
- 每周数据自动更新
- 手机端适配
- 离线缓存

## 数据来源

- 农业农村部
- 中国农业信息网
- 国家统计局

## 更新数据

### 方法1：运行更新脚本

1. 双击运行 `data_update.py`
2. 自动从网络获取最新数据
3. 重新部署到 Netlify

### 方法2：手动更新

1. 修改 `data.json` 文件中的价格数据
2. 重新部署到 Netlify

## 部署到 Netlify

1. 打开 https://app.netlify.com/drop
2. 拖入整个文件夹
3. 自动生成网站链接

## 文件说明

- `index.html` - 主页面
- `data.json` - 价格数据
- `data_update.py` - 数据更新脚本
- `manifest.json` - PWA配置
- `sw.js` - Service Worker

## 使用方法

1. 手机浏览器打开网站
2. 点击"添加到主屏幕"
3. 即可像App一样使用
